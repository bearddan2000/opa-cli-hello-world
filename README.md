# opa-cli-hello-world

## Description
A POC for opa programming language.

## Tech stack
- nodejs
    - npm
- java
- make
- ocaml

## Docker stack
- rzuckerm/opa

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
